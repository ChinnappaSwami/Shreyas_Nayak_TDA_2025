We know about the dfs & bfs traversals, but we have 4 more types of traversals when it comes to binary trees
1) Inorder   [ left node , root node,  right node ]
2) Postorder [ left node , right node, root node ]
3) Preorder  [ root node, left node , right node ]
4) Level order


Level order is just BFS.
Inorder, Postorder and Preorder are variants of DFS.
This is a very simple topic [if u understand BFS & DFS] and can be understood by just reading the articles given above once.
