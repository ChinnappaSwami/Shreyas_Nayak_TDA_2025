In recursions, sometimes it happens that you have to calculate smth, that u previously calculated in another recursion call. in simple terms the "computer forgot" that it calculated it, and calculates it again. this is kind of a redundancy and increases time. To reduce this one way is to "memorize" what u calculated. Basically store the outputs of each recursion call. 
This is your introduction to dynamic programming(dp), and is a very important concept used everywhere. This is asked a lot in interviews, and only practicing can improve ur ability in approaching questions using dp.
