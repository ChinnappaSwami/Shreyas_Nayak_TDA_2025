Dijkstras Algorithm is for finding the shortest path from a point to all other points. Time Complexity = O(Elog(V)) , E-> no. of Edges, V-> no. of vertices
Floyd warshalls algorithm is for finding shortest distance between any pair of points. Time Complexity = O(V^3)

The algorithms are not that tough and going through the above links and dry running the code with some random example to understand its concept should be enough.
